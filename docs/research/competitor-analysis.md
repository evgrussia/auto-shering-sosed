# Анализ конкурентов

P2P-каршеринг и коммерческий каршеринг в РФ. Цель: выявить нишу и функционал для конкурентного преимущества.

## 1. P2P-сегмент (прямые конкуренты)

### RentRide

| Параметр | Значение |
|----------|----------|
| Модель | Маркетплейс: частные владельцы + прокаты |
| География | РФ, 2000+ предложений, 1143+ авто в Москве |
| Цены | от 700 ₽ до 9000+ ₽/сутки по классам (эконом, комфорт, бизнес, премиум) |
| Комиссия | 15% с аренды |
| Верификация | Фото паспорта и ВУ; проверка по открытым/закрытым источникам (права, долги, судимости) |
| Для владельца | Сам тарифы, место передачи, календарь; доход 35–55 тыс. ₽/мес |
| Особенности | Рейтинг «Супервладельцы», доставка авто |

**Слабости / возможности для отличия:** нет фокуса на «соседях» и домовых чатах; нет умных замков; нет интеграции с Telegram; верификация не через Госуслуги; комиссия 15% — есть запас по цене/условиям.

## 2. Коммерческий каршеринг (косвенные конкуренты)

### Делимобиль

- ~30 000 авто, 16 городов, 12,6 млн пользователей; выручка 2024 ~27,9 млрд ₽.
- Тарифы: от 10,10 ₽/мин; сутки от 3699 ₽/24ч (150 км вкл.), далее 8 ₽/км.
- Страховка без франшизы на тарифе «Сказка».

### Яндекс Драйв

- Москва, СПб, Казань, Сочи и др.
- Тарифы: от 5 ₽/мин, 250 ₽/час, 2500 ₽/сутки; бензин, парковка, страховка входят.

### Ситидрайв (ex. Youdrive)

- 19 000+ авто; каршеринг, долгая аренда, прокат.

**Вывод:** коммерческий каршеринг — поминутно/почасово, сутки от ~2500–3700 ₽. P2P может быть на 30–50% дешевле при сохранении качества сервиса и доверия.

## 4. Выводы для конкурентного преимущества

1. **Верификация через Госуслуги** — дифференциация vs RentRide (ручная проверка документов); повышает доверие и снижает трение.
2. **Telegram WebApp + домовые чаты** — уникальная связка: уведомления в мессенджере, распространение через чаты ЖК/районов.
3. **Фокус «соседи»** — не просто P2P, а доверие через близость (один двор/район) и рейтинги; возможность кросс-промо в чатах.
4. **Умные замки (post-MVP)** — у RentRide нет; даёт «как в каршеринге» без передачи ключей и усиливает позицию.
5. **Арбитраж (фото до/после, претензии)** — явный процесс разрешения споров и рейтинги как доверие и защита владельца.
6. **Цена** — комиссия 20–25% при целевой цене для клиента на 30–50% ниже коммерческого каршеринга оставляет ценность и владельцу, и арендатору.

**Рекомендуемый приоритет фич для MVP и post-MVP:**  
MVP: бронирование + ручная передача ключей + верификация Госуслуги + рейтинги + задел под арбитраж.  
Post-MVP: умные замки, КАСКО-слот, оплата/экcцесс (ЮKassa), полный арбитраж (фото до/после).

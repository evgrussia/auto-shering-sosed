# Бизнес-модель

## 1. Canvas (упрощённый)

| Блок | Содержание |
|------|------------|
| **Ценностное предложение** | P2P-аренда авто между соседями: дешевле коммерческого каршеринга на 30–50%, верификация через Госуслуги, рейтинги и арбитраж; позже — умные замки и КАСКО. |
| **Клиентские сегменты** | Владельцы авто (монетизация простоя); арендаторы (короткая/долгая аренда без корпоративных наклеек). |
| **Каналы** | Telegram WebApp, домовые/районные чаты (бот или ссылка), сарафан, рейтинги. |
| **Взаимоотношения с клиентами** | Самообслуживание в приложении; поддержка при спорах (арбитраж); уведомления в Telegram. |
| **Источники дохода** | Комиссия платформы 20–25% с каждой оплаченной аренды (после ввода ЮKassa). Доп. доход (post-MVP): возможная маржа от страхового слота или партнёрств. |
| **Ключевые ресурсы** | Продукт (WebApp + API), интеграции (Госуслуги, ЮKassa, умные замки), бренд и доверие. |

| **Ключевые партнёры** | Госуслуги (API), ЮKassa, вендоры умных замков, (позже) страховые. |
| **Структура издержек** | Разработка, инфраструктура, комиссии платёжных систем, операционные расходы на поддержку и арбитраж. |

## 2. Монетизация

- **Основная:** комиссия **20–25%** от суммы аренды (удерживается при приёме оплаты через платформу).
- **Сравнение:** RentRide — 15%; целевая ценность для клиента (цена на 30–50% ниже коммерческого каршеринга) сохраняет место для комиссии 20–25%.
- **Этапы:** MVP без приёма денег платформой (оплата между сторонами или ручной учёт); после ввода ЮKassa — комиссия с каждой сделки.

## 3. Unit-экономика (целевая)

- **LTV владельца:** зависит от частоты сдачи и среднего чека; цель — удержание и рост числа бронирований.
- **LTV арендатора:** средний чек × частота аренд в год; повторные бронирования — ключевая метрика.
- **CAC:** привлечение через Telegram и домовые чаты снижает CAC; платная реклама — по мере масштабирования.
- **Комиссия 20–25%** должна покрывать операционные расходы, платежи, поддержку и арбитраж с запасом на маржинальность.

## 4. Масштабирование

- **География:** несколько городов с фокусом на листинги и локальное продвижение (чаты, сообщества).
- **Рост:** привлечение владельцев (больше предложений) → больше арендаторов → больше бронирований → больше комиссии; сетевой эффект внутри районов/чатов.
